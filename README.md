# Appium Ecommerce Automation Framework (WIP)

## Tech Stack

* Appium (Java Client)
* TestNG
* Maven

## Features (Current)

* Driver setup using UiAutomator2
* Config-driven execution (config.properties)
* Page Object Model (POM) implemented
* Login flow automated (Name, Gender, Country selection)
* Dynamic scrolling using UiScrollable
* Explicit waits for stability
* Clean project structure (Base, Pages, Tests, Utils)

## In Progress

* Product Page (Add to Cart flow)
* Gesture utilities (scroll, swipe, etc.)
* Reporting (Extent Reports)
* Data-driven testing
